# Specification

## Summary
**Goal:** Create a 2D endless runner game featuring Shinchan collecting chocobi chips while avoiding obstacles.

**Planned changes:**
- Implement side-scrolling 2D game with Shinchan character running continuously left-to-right
- Add chocobi chip collectibles that spawn randomly and increment score on collection
- Create obstacles (rocks and puddles) that trigger game over on collision
- Implement progressive difficulty with increasing speed based on score
- Support dual control system (keyboard arrow keys and mouse clicks)
- Display real-time score during gameplay and final score on game over screen
- Apply anime-inspired visual theme with playful colors and cartoon-style graphics
- Load all character and game assets from frontend/public/assets/generated directory

**User-visible outcome:** Players can control Shinchan in an endless runner game, collecting chocobi chips for points while avoiding obstacles, with the game becoming progressively faster as they score more points.
